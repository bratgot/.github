## ☕ Support This Work

[![Buy Me A Coffee](https://img.shields.io/badge/Buy%20Me%20a%20Coffee-☕-FFDD00?style=flat&logo=buymeacoffee&logoColor=black)](https://buymeacoffee.com/bratgot)

These tools are free and open source, but they aren't free to make. Behind
each release sits a lot of development time and real production hardware —
GPU-accelerated plugins don't test themselves.

Recent projects include:

- **C++ NDK plugins** for Nuke — compositing ops, geometry tools, and
  GPU-accelerated image processing with CUDA
- **On-GPU AI generation** inside Nuke — inpainting and image generation
  running locally, no render farm or cloud required
- **BlinkScript gizmos** — kernel-based tools for merging, vector blur,
  and other per-pixel work
- **Python pipeline tooling** — panels, bake systems, and workflow
  utilities for production compositing

If one of these tools saved you time on a shot, deadline, or delivery,
consider buying me a coffee. It helps cover development hardware, software
licenses, and the late nights that go into testing across Nuke versions.

No pressure — a star ⭐ or a bug report helps too.
